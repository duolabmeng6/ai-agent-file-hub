服务器部署

一键脚本会生成 Compose 配置。云服务器、私有服务器和现有 Docker 运维体系可以使用下面的命令。

docker compose 
git clone https://github.com/duolabmeng6/ai-agent-file-hub.git
cd ai-agent-file-hub
AGENT_FILE_HUB_VERSION=v1.0.0 HOST_PORT=18787 sh run.sh
docker run 
docker run -d --name agent_file_hub \
  --restart unless-stopped \
  -p 18787:9000 \
  -e PORT=9000 \
  -e GIN_MODE=release \
  -e FILE_BROWSER_ROOT=/app/storage \
  -v agent_file_hub_data:/app/data \
  -v "$PWD/storage:/app/storage" \
  duolabmeng/agent_file_hub:v1.0.0

系统要求

二进制部署适合轻量服务器和本机试用，Docker 部署适合生产环境。

Docker Docker 24+ / Compose v2 linux/amd64 / linux/arm64
Windows Windows 10 及以上 x64 / ARM64
macOS macOS 12 Monterey 及以上 Intel / Apple Silicon
Linux 主流 glibc 或 musl 发行版 x64 / ARM64
