File OS for Agents

AI智能体文件中枢

把本地目录、OSS/S3、SFTP、Git 仓库统一成一个面向人和 AI Agent 的文件、记忆与技能基础设施，并通过浏览器、REST、OpenAPI 与 MCP 访问同一套资产。

查看安装方法 下载 v1.0.0

最快部署

默认宿主机端口是 18787，容器内服务监听 9000。数据目录和默认本地储存器会持久化到 Docker volume。

Docker Hub
shell 
git clone https://github.com/duolabmeng6/ai-agent-file-hub.git
cd ai-agent-file-hub
AGENT_FILE_HUB_VERSION=v1.0.0 sh run.sh
访问地址 http://127.0.0.1:18787
镜像标签 duolabmeng/agent_file_hub:v1.0.0
持久化目录 /app/data 与 /app/storage

为 Agent 准备的文件基础设施

同一套文件资产可以服务人类界面、自动化脚本和 MCP 客户端。

Memory

长期记忆中心

用目录和 Markdown 承载项目上下文、知识库、Prompt、Workflow 与历史版本。

Storage

多储存器接入

统一接入本地目录、OSS、S3-compatible、SFTP 和 Git 仓库。

MCP

Agent 协作入口

共享目录可以生成 REST、OpenAPI 和 MCP Endpoint，让 Agent 安全访问工作区。
