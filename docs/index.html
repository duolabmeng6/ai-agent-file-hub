CLI · Agent / 脚本

afile 命令行模式

CLI 负责启动 Web 工作台、输出 MCP 配置和复用 Workspace 文件能力。储存器和协作目录先在 Web 中配置，终端只拿 workspace code 操作文件。

CLI 常用命令

~/agent-file-hub/afile serve --host 127.0.0.1 --port 9000 --background --open

~/agent-file-hub/afile mcp config <workspace-code> --server http://127.0.0.1:9000 --json && ~/agent-file-hub/afile workspace ls <workspace-code> /

给 Agent 使用的文件基础设施

一个入口管理文件、知识、技能资产和自动化访问，减少 AI Agent 在项目之间来回找上下文的成本。

Files

Memory

长期记忆中心

Agent

macOS / Linux

direct 安装会放置 agent_file_hub，当前版本包含 CLI 资产时会同时放置 afile，适合本机试用、脚本和 AI 技能调用。

curl -fsSL https://my.rongyiapi.com/ai-agent-file-hub/install.sh -o install.sh && AGENT_FILE_HUB_MODE=direct bash install.sh

curl -L -o agent_file_hub https://github.com/duolabmeng6/ai-agent-file-hub/releases/download/v1.0.2/agent_file_hub-linux-amd64 && chmod +x agent_file_hub && PORT=18787 FILE_BROWSER_ROOT=./storage ./agent_file_hub

curl -L -o agent_file_hub https://github.com/duolabmeng6/ai-agent-file-hub/releases/download/v1.0.2/agent_file_hub-darwin-arm64 && chmod +x agent_file_hub && PORT=18787 FILE_BROWSER_ROOT=./storage ./agent_file_hub
